# 🚴 ESP32 Bicycle Computer

Developed a custom bicycle computer system with real-time velocity and distance tracking, including an Android application. 
Designed and implemented embedded firmware in C++ for data acquisition and processing on the hardware device, 
and built an Android app in Kotlin to display, log, and visualize ride statistics. Created custom PCB layouts and s
chematics to support the hardware design and demonstrate full hardware–software integration.

## Features

- Real-time speed measurement  
- Distance tracking  
- Ride time monitoring  
- OLED display interface (SSD1306)
- Android application

## System Overview

- ESP32 – main controller  
- SSD1306 OLED – display
- Hall sensor

## 🛠️ Technologies

- C/C++ (Arduino)
- ESP32
- Kotlin (Android Studio)

## 🚀 Getting Started

### Hardware

- ESP32  
- SSD1306 OLED
- Hall sensor  

### Setup

1. Clone repo
   
git clone https://github.com/AtlagicBartol/Bicycle_computer.git


3. Open in Arduino IDE  

4. Install libraries:
- Adafruit SSD1306  
- Adafruit GFX  

4. Upload to ESP32
